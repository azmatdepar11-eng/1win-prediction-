# 1win-pred-bot-demo
Demo Telegram bot (educational) — simulated predictions for Aviator, Crash, Roulette, Mines.
**IMPORTANT**: Demo-only. No real account access. Check local laws & platform ToS before real money use.

## Quick start (Replit)
1. Create a new Repl (Import from GitHub or create new Python Repl).
2. Add files: `bot_strong_games.py`, `requirements.txt`, `.replit`.
3. Set Replit secret `BOT_TOKEN` to your Telegram bot token.
4. Press Run. Open Telegram bot and send `/start`.
